Community Collaboration Tools

Web-based tools for TrueSight DAO members to manage cacao inventory, participate in the Sunmint tree planting program for carbon credits, handle voting rights, report community contributions, and notarize official documents.

Quick Access

DAO Contribution Reporter

Report volunteer time or out-of-pocket expenses for DAO operations.

Features:

  • Select contributors and contribution type
  • Specify amount and description
  • Attach proof (image/PDF) and share signed report

Usage:

  1. Open DAO Contribution Reporter.
  2. Select contributors, type, and amount.
  3. Submit report with proof to our Telegram channel.

Inventory Expense Reporter

Report DAO inventory expenses.

Features:

  • Select resources and quantities
  • Attach proof (image/PDF)
  • Share expense reports

Usage:

  1. Open Expense Reporter.
  2. Select resource and quantity.
  3. Submit report with proof to our Telegram channel.

Inventory Movement Reporter

Track inventory transfers.

Features:

  • Select items and recipients
  • Capture snapshots
  • Share transfer reports

Usage:

  1. Open Movement Reporter.
  2. Select item and recipient.
  3. Submit report with snapshot to our Telegram channel.

Batch QR Code Generator

Generate multiple QR codes for inventory items in bulk.

Features:

  • Select currency and quantity
  • View available quantities across all ledgers
  • Generate batch QR codes with digital signature
  • Receive email notification with zip file download link

Usage:

  1. Open Batch QR Code Generator.
  2. Select currency and specify quantity (1-100).
  3. Submit request with your digital signature.
  4. Check your email for the zip file download link.

Stores Nearby

Find the top 10 stores nearest to your location from the holistic wellness hit list.

Features:

  • Filter stores by status (Contacted, Research, Partnered, etc.)
  • Use your current location or enter coordinates manually
  • View stores ordered by distance
  • Click on any store to view full details

Usage:

  1. Open Stores Nearby.
  2. Select status filter (default: Contacted).
  3. Click "Use Current Location" or enter latitude/longitude manually.
  4. Click "Find Nearby Stores" to see the top 10 nearest stores.
  5. Click on any store card to view full details.

Stores by Status

Browse Hit List rows filtered by status (and optional shop type), then open Store Interaction History for a store.

Usage:

  1. Open Stores by Status.
  2. Optional: select one or more statuses / shop types (leave empty for all).
  3. Click Refresh list after spreadsheet updates.
  4. Click History on a row to review interaction history.

Store Interaction History

Review holistic wellness Hit List context (follow-ups, DApp remarks, email agent tabs) before partner outreach.

Features:

  • Search stores by name or address
  • See remarks, follow-up log, and suggestion rows when configured

Usage:

  1. Open Store Interaction History.
  2. Search and expand rows as needed.

Outbound Review

Triage pending Gmail drafts (warm-ups and follow-ups): lint flags risky ones (generic info@, fallback shop name, foreign-script venue, etc.), Hosts Circles=Yes prospects get a blue badge, clean cohort goes to the bottom for batch send. Tabs at top toggle between cohorts.

Features:

  • Mobile-first scan view sorted red-first
  • Tap a count chip (red / yellow / clean / Hosts Circles) to filter the list
  • Tap Send to ship the draft from your Gmail (signed via [WARMUP SEND EVENT])
  • Tap Edit in Gmail to deep-link into the draft for hand-editing first

Usage:

  1. Open Outbound Review.
  2. Pick the Warm-ups or Follow-ups tab.
  3. Scan top-to-bottom; reds + Hosts Circles float up. Tap a count chip to filter.
  4. Tap Send for trusted drafts; tap Edit in Gmail for ones you want to tweak.

Update QR Code

Update QR code status, email address, and associated member.

Features:

  • Select QR code from dropdown list
  • View and update current status (ACTIVE, SOLD, MINTED, CONSIGNMENT)
  • Update owner email address
  • Associate QR code with a member/manager
  • Current values are pre-populated for easy review

Usage:

  1. Open Update QR Code.
  2. Select a QR code from the dropdown (or search by typing).
  3. Review current values that are automatically loaded.
  4. Modify status, email, and/or member association as needed.
  5. Submit the update with your digital signature.

Repackaging Planner

Load inventory by holder (same API as Inventory Movement), add input lines with quantities, optional conversion fee (e.g. Santos/Martinus/CIC), and preview a suggested Currency name plus cost per output unit.

Features:

  • Loads Main Ledger + managed AGL lines for the holder (optional main-only for speed)
  • Inventory holder vs “booked by” naming
  • Producer suffix presets (Santos, Martinus, CIC, San Francisco - AGL4)
  • Custom lines with manual unit costs for AGL inventory

Usage:

  1. Open Repackaging Planner.
  2. Select holder, load inventory, add input rows.
  3. Set outputs, fee, and naming fields; click Calculate.

Restock Recommender

Get the optimal number of bags to send a partner from Kirsten's (SF), using prior sales and shipping cost.

Features:

  • Select partner and view prior sales data
  • Factor in shipping cost per bag
  • Get recommended restock quantity
  • Copy formatted reply for sharing

Usage:

  1. Open Restock Recommender.
  2. Select the partner from the dropdown.
  3. Click "Get Recommendation" and review the suggested quantity.
  4. Copy the reply text to share with the partner.

Shipping Planner

Select inventory items and calculate shipping costs for local (USPS via EasyPost) or freight shipping.

Features:

  • Select member/manager and inventory items
  • Choose shipping type (Local or Freight)
  • Enter destination address for local shipping
  • Calculate shipping cost estimates
  • Save shipping manifest as PDF

Usage:

  1. Open Shipping Planner.
  2. Select member and inventory items.
  3. Choose packaging (Box or Pallet) and enter destination.
  4. Click "Calculate Shipping Cost" to get the estimate.
